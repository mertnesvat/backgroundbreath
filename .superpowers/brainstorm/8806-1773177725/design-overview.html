Circle pulses 5.5s inhale → 5.5s exhale. Click-through, always on top. Warm glow at ~15% opacity in production.

Architecture — 5 components

BreathOverlayApp

@main SwiftUI App. Bridges to AppDelegate via @NSApplicationDelegateAdaptor. Sets LSUIElement = true (no dock icon).

BreathOverlayApp.swift

AppDelegate

NSApplicationDelegate. Creates NSStatusItem (menu bar icon). Shows/hides/quits. Owns the FloatingPanel lifecycle.

AppDelegate.swift

FloatingPanel

NSPanel subclass. .floatingWindowLevel, non-activating, click-through (ignoresMouseEvents = true), transparent background, draggable.

FloatingPanel.swift

BreathTimer

ObservableObject. Drives inhale/exhale phase. Timer fires every 5.5s, toggles @Published phase enum. Pause/resume support.

BreathTimer.swift

BreathAnimationView

SwiftUI View hosted in FloatingPanel via NSHostingView. Warm amber/orange radial gradient orb (~60pt). Scales 0.6→1.0 with easeInOut over 5.5s, synced to BreathTimer phase. Background clear, no window chrome.

BreathAnimationView.swift

Menu bar actions (V1)

● Start / Pause — toggles BreathTimer
● Quit — NSApp.terminate
